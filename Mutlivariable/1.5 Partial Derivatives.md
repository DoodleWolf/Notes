# 1.5.1 Definition (Partial Derivatives)
For the function $f(x,y)$
$$ \text{ Slope in x direction } = \left( \frac{\partial f}{\partial x} \right) _{y= \text{constant}} = \lim _{h \rightarrow 0} \left\{ \frac{f(x+h,y) - f(x,y)} {h} \right\}$$
