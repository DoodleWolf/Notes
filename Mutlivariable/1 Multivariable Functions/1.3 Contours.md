# Contour 
## Definition
A level set of a graph
![[Pasted image 20210202180105.png]]
$$L_{c}=L_{c}(f)=\{(x,y) \in \mathbb{R}^{2}| f(x,y)=c \}$$

## Examples
$$f(x,y)=1-x-y$$
Level sets are given by $c=f(x,y)$
